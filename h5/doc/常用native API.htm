常用native API 文档

创意云-移动端

 

本文档整理了JS-Native调用中，Native部分的常用API，基本和硬件、手机功能相关

 

API常量名及业务返回码范围

（统一返回码另算）

1.   打开GPS并定位

NATIVE_FUNCTION_OPENGPS

业务返回码范围20000-24999

2.   打开拨号盘并输入号码（拨打需要用户触发）

NATIVE_FUNCTION_CALLTEL

ret-range:25000-29999

3.   将内容复制到剪切板

NATIVE_FUNCTION_SENDTOCLIPBOARD

ret-range:30000-34999

4.   获取剪切板的最后一条内容

NATIVE_FUNCTION_GETFROMCLIPBOARD

ret-range:35000-39999

5.   打开消息并填写收件人、短信内容（发送需要用户触发）

NATIVE_FUNCTION_SENDMESSAGE

ret-range:40000-44999

6.   打开email并填写收件人、email内容（发送需要用户触发）

NATIVE_FUNCTION_SENDEMAIL

ret-range:45000-49999

7.   打开摄像机扫描二维码，条形码等

NATIVE_FUNCTION_OPENCAMERA_SCAN

ret-range:50000-54999

 

1.    打开GPS并定位

获取定位信息，在GPS不可行时会使用WIFI、基站等手段定位

参数列表

null

返回参数

字段名

说明

latitude

纬度

longitude

经度

time

毫秒级时间戳

radius

定位精度 m

 

返回参数示例

{

    “ret”:”200”,

    “msg”:”it works okey”,

    “data”:{

       “latitude”:”12.321”,

       “longitude”:”34.567”,

        “time”:”1456196624000”,

        “radius”:”50.12345”

    }

}
